# Foto Amplificatori — Configuratore Casse

Carica qui le foto **originali e licenziate** dei moduli amplificatori.
Finché un file manca, il configuratore mostra automaticamente un'illustrazione
SVG generata dai dati del componente — nessun riquadro vuoto.

## Formato consigliato
- Estensione: `.webp` (oppure aggiorna il campo `image` in `src/data/speakerDatabase.ts`)
- Sfondo: trasparente o scuro (il tema del sito è scuro)
- Dimensione: ~1000×560px, orizzontale (formato 16:9), modulo centrato

## Nomi file attesi (devono combaciare con `speakerDatabase.ts`)

| File | Amplificatore |
|------|---------------|
| `hypex-fa123.webp` | Hypex FA123 |
| `hypex-fa252.webp` | Hypex FA252 |
| `hypex-fa502.webp` | Hypex FA502 |
| `icepower-125asx2.webp` | ICEpower 125ASX2 |
| `icepower-250asx2.webp` | ICEpower 250ASX2 |
| `powersoft-digimod1000.webp` | Powersoft DigiMod 1000 |
| `powersoft-digimod3000.webp` | Powersoft DigiMod 3000 |
| `pascal-spro2-500.webp` | Pascal S-PRO2 500 |
| `icepower-50asx2-bt.webp` | ICEpower 50ASX2 BT |

> ⚠️ Usa solo immagini di cui possiedi i diritti d'uso (acquistate o fornite
> dal produttore/distributore per la rivendita).
